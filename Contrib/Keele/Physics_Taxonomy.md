# Taxonomy for Physics Education
_a work in progress_ - Boyd Duffee, 2017-2019

This file provides a starting point for developing a taxonomy
for Physics to mirror the one available for Mathematics in the OPL.
These are the notes that I created while considering how problems
could be classified in a hierarchy.  It has not yet gone through
a review process, so the structure is still flexible.
Comments on the taxonomy can be made by raising an Issue on
[my cloned Github repository](https://github.com/duffee/webwork-open-problem-library/issues)

The guiding principles for developing this Taxonomy
* Maximum of **Three Levels Deep**
* Easily discoverable using the **Principle of Least Surprise**
* Must have **No Omissions**
* Try to have **No Duplications**
